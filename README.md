## 📱 Overview

MoneyFlow is a modern personal finance management application built with Flutter.

Track income, expenses, transfers, account balances, savings, and financial reports through a clean, fast, and user-friendly interface.

The application is designed with an offline-first architecture, ensuring all financial data remains on the user's device while offering powerful reporting, backup & restore, AdMob monetization support, remote configuration, and GitHub-based in-app updates.

## 🔐 Privacy

MoneyFlow follows an offline-first approach.

* No Login Required
* No Cloud Storage Required
* Data Remains on Device
* Full User Control Over Backups
* No Financial Data Stored on External Servers
